## PARTE 9: PROBLEMAS Y SOLUCIONES

### Problema 1: "No puedo cobrar en USD desde Cuba"

**Soluciones:**
```
1. Upwork/Toptal te pagan a cuenta bancaria internacional
   - Si no tienes: Pide a familiar en EEUU
   - O usa remesador local

2. Cliente te envía dinero directo a EEUU
   - Tú negocias cómo
   - Crypto, transferencia, efectivo

3. Agencia te contrata como "contractor"
   - Ellos manejan impuestos
   - Tú recibes dinero limpio
```

### Problema 2: "Qué pasa si pierdo cliente freelance?"

**Solución:**
```
❌ Mal: Depender de 1 cliente
✅ Bien: Tener 3-5 clientes pequeños
   - Si pierdes 1 = sigues con 4
   - Volumen diversificado = seguro

Estrategia:
- Año 1: 3-5 clientes, 10-15h/semana
- Año 2: 4-8 clientes, 15-20h/semana
- Año 3: 5-10 clientes, estable
```

### Problema 3: "Cómo compito con developers de EEUU?"

**Solución:**
```
NO compites por precio
Compites por:
✅ Nicho expertise (Flutter + Firebase specialist)
✅ Calidad código
✅ Rápido delivery
✅ Good communication
✅ Buenas reviews (5 estrellas)

Estrategia:
- Primeros 3 proyectos: $30/hora (build portfolio)
- Proyectos 4-10: $40/hora (con rating)
- Proyectos 11+: $50/hora (especialista)
```
