## PARTE 1: CONTEXTO ECONÓMICO CUBA 2026

### Realidad Económica

```
MONEDA:
- CUP (Peso Cubano) = 1 USD ≈ 350-400 CUP (paralelo)
- Salario promedio: 3,000-5,000 CUP/mes
- En USD: $7.50-12.50/mes

MERCADO:
- Población: ~11 millones
- Salones privados: ~5,000-8,000 (estimado)
- Acceso a internet: ~40-50%
- Datos móviles: Caro (CUCube ~$0.40/MB)
- Poder de compra: MUY bajo

SALONES PRIVADOS (Realidad):
- La mayoría: 1-2 personas (dueño + 1 trabajador)
- Ingresos diarios: 500-2,000 CUP
- Ingresos mensuales: 10,000-50,000 CUP ($25-125 USD)
- Margen neto: ~20-30%
- Ganancia mensual real: 2,000-15,000 CUP ($5-40 USD)
```

### Competencia Actual

```
Situación REAL:
❌ Casi NINGÚN salón usa software (aún)
✅ 95% usan: Libreta de papel + lápiz
✅ Algunos: Calculadora + cuaderno
❌ Muy pocos: Teléfono simple (no smartphone)

Razones:
1. Datos móviles MUY caros
2. Teléfonos inteligentes no son comunes
3. No hay cultura de software
4. Ingresos demasiado bajos para invertir
```

## PARTE 7: REALIDADES QUE DEBES SABER

### Barrera 1: Acceso a Internet

```
Problema: No todos tienen datos ilimitados
Solución: 
❌ Modo offline NO viable (requiere backend)
✅ App debe usar POCO datos (<1 MB/mes por usuario)

Impacto en adopción:
- Sin conexión confiable: Pierdes 20-30% de mercado
```

### Barrera 2: Dispositivos

```
Realidad Cuba:
- Teléfono smartphone: NO es común
- Tablet: Muy raro
- PC/Laptop: Algunos salones

Solución:
✅ App WEB (Chrome) = mejor que nativa
✅ Responsive: Debe funcionar en cualquier pantalla

Impacto:
- Web version = +40% adopción
```

### Barrera 3: Método de Pago

```
Métodos disponibles en Cuba:
❌ Tarjeta de crédito: Bloqueo EEUU (imposible)
❌ PayPal: No funciona en Cuba
❌ Transferencia bancaria: Muy complicada
❌ Criptomonedas: Solo usuarios tech

✅ Posibles:
- Transferencia de dinero en efectivo
- Pago local (otro cubano en EEUU)
- Préstamo de tarjeta (de amigo)
- Sistema de crédito manual

Realidad:
Pagos muy difíciles = ADOPCIÓN BAJA
```
